Spanning Tree

Necessary graph state:
A spanning tree is implemented on connected, undirected graphs. (Graphs can be weighted or non-weighted.)

User input required:
You select a starting node.

Description:
Beginning at a specified starting node, a spanning tree starts with an edge incident to that node and includes the least number of edges necessary in order to connect all nodes within the graph. Edges are selected to be in the spanning tree under the stipulation that the edges are incident to already visited nodes, (beginning at the start node), as long as no cycles are produced by selection of any edge. More than one spanning tree may exist for the same starting node on the same graph.
(Similar algorithm is Minimum Spanning Tree)
