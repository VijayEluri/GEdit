Connected Component Identification

Necessary graph state:
Connected component identification can be done on any type of graph.

User input required:
None

Description:
The Identify Connected Components algorithm finds and returns all islands in a graph. Due to the nature of this graph display tool, no explicit call to run this algorithm is necessary. Distinct islands are always displayed in separate windows at all times. This is true whether you are creating a graph, editing an existing graph, or executing an algorithm. For this reason, you are always aware of all of the islands for any given graph.
