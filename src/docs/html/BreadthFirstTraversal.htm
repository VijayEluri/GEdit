Breadth-first Traversal

Necessary graph state:
A breadth-first traversal is implemented on undirected graphs. (Graphs can be weighted or non-weighted.)

User input required:
You select a starting node.

Description:
This algorithm returns all possible vertices of the graph that are reachable from the given starting node. The vertices are returned in a sequence that starts by reaching to all nodes adjacent to the starting node in any direction. Then it continues from each of the adjacent nodes to visit all nodes immediately next to each, and so on, visiting all nodes that have not yet been visited. Execution stops when the algorithm can reach no more non-visited nodes. More than one breadth-first traversal may exist for the same starting node in the same graph.
