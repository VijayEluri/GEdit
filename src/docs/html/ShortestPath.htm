Shortest Path

Necessary graph state:
A shortest path is implemented on weighted, directed graphs.

User input required:
None

Description:
The shortest path algorithm returns the shortest path between two nodes, a starting node and a destination node. It ensures that the path chosen has the minimal total weight possible. More than one shortest path may exist between the same two nodes.
(Similar algorithm is All-Pairs Shortest Paths)
