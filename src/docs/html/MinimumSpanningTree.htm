Minimum Spanning Tree

Necessary graph state:
A minimum spanning tree is implemented on connected, directed, weighted graphs.

User input required:
None

Description:
A minimum spanning tree acts like a spanning tree, in that it returns the smallest number of edges such that all nodes in a graph are connected by those edges, and where no cycles exist. The minimum spanning tree adds the further requirement that the weights of the edges included in the tree have the lowest total weight possible. A graph may have more than one minimal spanning tree.
(Similar algorithm is Spanning Tree)
