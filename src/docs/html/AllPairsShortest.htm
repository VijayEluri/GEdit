All-pairs Shortest Paths

Necessary graph state:
The all-pairs shortest paths algorithm is implemented on weighted, directed graphs.

User input required:
None

Description:
All-pairs shortest path returns the shortest path(s) between every possible pairing of nodes in a given graph. (See Shortest Path algorithm description.) More than one all-pairs shortest path solution may be possible on a given graph.
